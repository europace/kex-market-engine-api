### Response

Antwort (Http Status Code 500) bei einem technischem Fehler bei der Angebotsannahme.

```json
{
    "supportMeldung": "ersetze diese Meldung durch eine hilfreiche Meldung für den Support"
}
```

oder auch

```json
{
    "angebot": null,
    "supportMeldung": "ersetze diese Meldung durch eine hilfreiche Meldung für den Support"
}
```
